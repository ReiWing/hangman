Требования для виселицы

1. Ввести слово
  a. Показать текст: "Enter new word" [keyboard.EnterWord]
  b. Сохранить каждый символ в массив
2. Сгенерировать слово
3. Задать количество попыток
4. Показывать сколько букв в слове: "_ _ _ _ _ _ _ _"
5. Вводить буквы для угадывания
6. Показывать использованные и неиспользованные буквы после ввода
7. Показывать оставшиеся попытки
9. Выдавать сообщение после ввода, уже использованных букв
10. Выдавать сообщение при победе
11. Выдавать сообщение при проигрыше
